P7 — SKETCH CREATE (variants we missed on the first pass)
[ ] D40. 2-Tangent Circle
         - Pick 2 sketch lines, specify radius → circle tangent to both lines.
         - Fusion: "creates a circle tangent to 2 sketch lines."
         - Add 'circle-2tangent' Tool + picker flow.

[ ] D41. 3-Tangent Circle
         - Pick 3 sketch lines → circle tangent to all three (incircle).
         - Add 'circle-3tangent' Tool + incircle solver.

[ ] D42. Line-tool click-drag → tangent arc
         - Fusion's Line tool: click to place points for straight segments,
           click-and-drag from a previous endpoint to create a connected
           tangent arc. Core chaining behavior; users rely on it constantly.
         - Add pointerdown/move/up state machine in Viewport.tsx
           SketchInteraction so Line can chain arcs.

[ ] D43. Midpoint Line
         - Fusion has a "Create midpoint lines in a sketch" subtopic — line
           defined by midpoint + one endpoint, auto-applies a Midpoint
           constraint so it stays symmetric.

[ ] D44. Blend Curve
         - Tangent (G1) or curvature-continuous (G2) blend curve between
           endpoints of two existing sketch curves / edges. Lives in Modify.

[ ] D45. Include 3D Geometry
         - Distinct from Project: pulls 3D edges / work geometry / sketch
           curves into the active sketch as-is (no projection onto plane).
         - Needed for 3D sketches + sweep paths.

[ ] D46. Project to Surface
         - Project sketch curves, edges, vertices, or work points onto a
           selected face (result wraps the face, not the sketch plane).

[ ] D47. Sketch Chamfer — 3 variants
         - Equal Distance Chamfer — one distance applied to both sides.
         - Two Distance Chamfer — separate distance per side.
         - Distance and Angle Chamfer — distance + angle on one side.
         - Replaces the straight-corner vertex with a chamfer line; same
           pick flow as Fillet but produces a line instead of an arc.
         - Add 'sketch-chamfer-equal' / '-two-dist' / '-dist-angle' tools.

[ ] D48. Arc Length dimension
         - Missing from D28's dimension list. Pick an arc → dimension becomes
           its arc length rather than radius.

[ ] D49. Aligned (true-length) dimension
         - D28 lists linear/angular/radial/diameter but misses Aligned: pick
           2 points and the dimension is measured along the line between
           them, not along X or Y axes.

[ ] D50. AutoConstrain
         - Fusion's SKT-AUTO-CONSTRAIN: analyzes the current sketch, proposes
           constraints + dimensions in purple, lets user tweak via a slider
           (how aggressive) and selectively apply. Depends on the solver from
           D27 but also needs a candidate-generation pass (near-parallel,
           near-horizontal, near-equal-length detection with tolerances).

[ ] D51. Curvature constraint (G2)
         - Not in D29's list. Constrains a spline and another sketch curve
           to share curvature (smooth continuous G2 blend). Only meaningful
           once splines (D9) ship.

[ ] D52. Symmetry constraint (explicit)
         - Already listed as a geometric constraint in D29 but Fusion's
           Mirror tool AUTO-APPLIES symmetry. When D21 (sketch mirror) lands,
           every mirrored pair must get a symmetry constraint so dragging
           one side drags the other. Track as a separate acceptance check.

[ ] D57. Linetype conversion post-hoc
