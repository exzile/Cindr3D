 DESIGN TEST: Orange Pi 3 LTS Case
 Created: 2026-05-02
 Purpose: Step-by-step reproduction guide for creating an OPi3 LTS case

REFERENCE DIMENSIONS (from orangepi.org):
  PCB:            56mm x 85mm
  Wall thickness:  2mm
  Case clearance:  1mm per side (interior = 58mm x 87mm)
  Case exterior:   62mm x 91mm (with 2mm walls)
  Case height:     25mm (bottom) + lid
  Lid height:       5mm
  Mounting holes:   M3 (3.2mm diameter), 4 corners
  Mounting hole inset: 3.5mm from edges
  Corner fillets:   3mm exterior, chamfers on lid edges

CONNECTOR CUTOUTS (one short edge — "back"):
  Ethernet (RJ45):  W=16mm, H=13.5mm, centered at ~14mm from left edge
  HDMI:             W=15mm, H=6.5mm,  centered at ~32mm from left edge
  USB 3.0:          W=14.5mm, H=8mm,  centered at ~50mm from left edge

CONNECTOR CUTOUTS (other short edge — "front"):
  USB-C power:      W=9mm,  H=3.5mm,  centered at ~10mm from left edge
  USB 2.0 (x2):     W=14.5mm, H=8mm,  centered at ~35mm from left edge (stacked)

SD CARD SLOT (long edge):
  W=13mm, H=2.5mm, positioned ~5mm from back edge

GPIO (long edge, opposite side):
  No cutout needed (internal header)

VENTILATION:
  Top lid: grid of vent holes or slot pattern

 STEPS
===============================================================================

--- PHASE 1: PROJECT SETUP ---

[ ] STEP 1: Open Cindr3D application (localhost)
    - Navigate to localhost:5173 in browser
    - Verify the app loads with empty workspace

[ ] STEP 2: Create root component "OPi3_Case"
    - Click "Add Component" in the component tree
    - Name it "OPi3_Case"

[ ] STEP 3: Create sub-component "Bottom"
    - Under OPi3_Case, add sub-component "Bottom"

[ ] STEP 4: Create sub-component "Lid"
    - Under OPi3_Case, add sub-component "Lid"

--- PHASE 2: BOTTOM CASE — BASE SKETCH ---

[ ] STEP 5: Create new sketch on Bottom component
    - Select Bottom component
    - Click "New Sketch" on XY plane
    - Enter sketch mode

[ ] STEP 6: Draw outer rectangle (case exterior)
    - Use Rectangle tool
    - Draw rectangle: 62mm x 91mm
    - Use Dimension tool to dimension width = 62mm
    - Use Dimension tool to dimension height = 91mm
    - Verify dimensions appear on sketch

[ ] STEP 7: Draw inner rectangle (hollow interior)
    - Use Rectangle tool
    - Draw rectangle: 58mm x 87mm, offset 2mm from outer edges
    - Dimension width = 58mm
    - Dimension height = 87mm

[ ] STEP 8: Add mounting post circles (4 corners)
    - Switch to Circle tool
    - Draw 4 circles at mounting hole positions:
      * (5.5, 5.5) — r=3mm outer post
      * (5.5, 85.5) — r=3mm
      * (56.5, 5.5) — r=3mm
      * (56.5, 85.5) — r=3mm
    - Dimension each circle radius = 3mm

[ ] STEP 9: Add mounting screw holes (4 circles inside posts)
    - Draw 4 smaller circles at same centers:
      * r=1.6mm (M3 screw clearance)
    - Dimension each = 1.6mm radius

[ ] STEP 10: Commit sketch
    - Exit sketch mode
    - Verify sketch appears in tree

--- PHASE 3: BOTTOM CASE — EXTRUDE ---

[ ] STEP 11: Extrude base
    - Select the outer rectangle profile
    - Extrude 2mm for the solid base floor

[ ] STEP 12: Extrude walls
    - Select the wall ring (between outer and inner rectangles)
    - Extrude 25mm upward for walls

[ ] STEP 13: Extrude mounting posts
    - Select mounting post ring profiles (between 3mm and 1.6mm circles)
    - Extrude 7mm upward (standoff height for PCB)

[ ] STEP 14: Undo/Redo test
    - Undo the last extrude (Ctrl+Z)
    - Verify it reverts cleanly
    - Redo (Ctrl+Y)
    - Verify it restores correctly
    - NOTE any bugs found here

--- PHASE 4: CONNECTOR CUTOUT SKETCHES ---

[ ] STEP 15: Create sketch on back wall face (short edge, Z-up)
    - Select the back wall face
    - Create new sketch

[ ] STEP 16: Draw Ethernet cutout rectangle
    - Rectangle: W=16mm, H=13.5mm
    - Position: centered at 14mm from left, 9mm from bottom (PCB at 7mm + 2mm base)
    - Dimension W=16, H=13.5

[ ] STEP 17: Draw HDMI cutout rectangle
    - Rectangle: W=15mm, H=6.5mm
    - Position: centered at 32mm from left, 9mm from bottom
    - Dimension W=15, H=6.5

[ ] STEP 18: Draw USB 3.0 cutout rectangle
    - Rectangle: W=14.5mm, H=8mm
    - Position: centered at 50mm from left, 9mm from bottom
    - Dimension W=14.5, H=8

[ ] STEP 19: Commit back-wall sketch
    - Exit sketch

[ ] STEP 20: Cut-extrude through back wall
    - Select all cutout profiles
    - Extrude-cut through 2mm wall
    - Verify holes appear

[ ] STEP 21: Undo/Redo test on cut
    - Undo cut (Ctrl+Z)
    - Verify wall is solid again
    - Redo (Ctrl+Y)
    - Verify cutouts return
    - NOTE any bugs

--- PHASE 5: FRONT WALL CUTOUTS ---

[ ] STEP 22: Create sketch on front wall face
[ ] STEP 23: Draw USB-C cutout (W=9, H=3.5)
    - Dimension both
[ ] STEP 24: Draw USB 2.0 cutout (W=14.5, H=8)
    - Dimension both
[ ] STEP 25: Commit and cut-extrude front wall
[ ] STEP 26: Undo/Redo test

--- PHASE 6: SIDE WALL CUTOUT (SD CARD) ---

[ ] STEP 27: Create sketch on side wall face (long edge)
[ ] STEP 28: Draw SD card slot cutout (W=13, H=2.5)
    - Dimension both
[ ] STEP 29: Commit and cut-extrude
[ ] STEP 30: Undo/Redo test

--- PHASE 7: FILLETS AND CHAMFERS ON BOTTOM ---

[ ] STEP 31: Apply 3mm fillet to all 4 outer vertical edges
    - Select each vertical edge
    - Apply fillet radius = 3mm
    - Verify smooth corners

[ ] STEP 32: Apply 1mm chamfer to top rim of walls
    - Select top edges
    - Apply chamfer = 1mm
    - This creates a lead-in for the lid

[ ] STEP 33: Undo/Redo test on fillets
    - Undo fillet
    - Verify sharp corners return
    - Redo
    - Verify fillets return
    - NOTE any bugs

--- PHASE 8: LID ---

[ ] STEP 34: Switch to Lid component
    - Select Lid in component tree

[ ] STEP 35: Create sketch on XY plane (at Z=25mm or top of case)
    - New sketch

[ ] STEP 36: Draw lid outer rectangle
    - 62mm x 91mm (same as case exterior)
    - Dimension W=62, H=91

[ ] STEP 37: Draw lid inner lip rectangle
    - 58mm x 87mm (fits inside case walls)
    - Dimension W=58, H=87

[ ] STEP 38: Draw vent hole pattern on lid
    - Array of circles or slots for ventilation
    - Dimension representative holes

[ ] STEP 39: Commit lid sketch

[ ] STEP 40: Extrude lid body
    - Extrude outer profile 3mm (lid thickness)
    - Extrude inner lip 2mm downward (press-fit into case)

[ ] STEP 41: Apply fillets to lid
    - 3mm fillet on outer vertical edges (match case bottom)
    - 0.5mm chamfer on top edges

[ ] STEP 42: Undo/Redo test on lid

--- PHASE 9: COMBINE / FINAL CHECKS ---

[ ] STEP 43: Use Combine tool if needed
    - Combine mounting posts with base body (union)
    - Verify single solid body per component

[ ] STEP 44: Final undo/redo stress test
    - Undo several operations
    - Redo all
    - Verify model integrity

[ ] STEP 45: Verify component tree
    - OPi3_Case (root)
      ├── Bottom (body with cutouts, fillets, posts)
      └── Lid (body with lip, vents, fillets)
